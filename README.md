# App para Aprender Multiplicación

Una app ligera para practicar multiplicaciones con ejercicios adaptativos,
exploración de la tabla, rachas y progreso local.

## Ejecutar

Abre `index.html` en un navegador.

No hace falta instalar dependencias.

## Funciones

- Modo de práctica con feedback inmediato
- Factores seleccionables del 1 al 12
- Sesiones con cronómetro o sin límite de tiempo
- Progreso guardado en `localStorage`
- Tabla de multiplicar con precisión por factor
- Repaso que prioriza los ejercicios fallados
